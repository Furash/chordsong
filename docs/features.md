# Features

Discover the powerful capabilities of Chord Song.

## Overview

Chord Song provides a powerful chord-based navigation system for Blender. Pressing the leader key opens the overlay, which lists all available chords for the current context. As you type letters, the list filters to show only matching sequences.

## Available Features

- **[Info Panel](features/info.md)**: Batch-create chord mappings from Blender's operator/property history
- **[Recents System](features/recents.md)**: Quick access to recently used chords
- **[Conflict Detection](features/conflicts.md)**: Identify and resolve chord mapping conflicts
- **[Context Menu Integration](features/context_menu.md)**: Create mappings directly from Blender's context menu

## Integration

Chord Song works across various Blender editors:
- 3D Viewport
- Shader Editor
- UV Editor
- Node Editor
- Image Editor
- And more!
