## Property Mappings

Map chords to Blender properties.

### Overview

Property mappings allow you to set or toggle Blender properties directly.

### Creating Property Mappings

You can create property mappings by:
- Right-clicking on properties in Blender's UI
- Using the "Add Mapping" button in preferences
- Extracting from the Info panel - see [Info Panel workflow](../../features/info.md) for batch creation from property assignments
