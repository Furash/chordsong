# Overlay Configuration

Configure the overlay appearance and behavior.

## Overview

The overlay is designed to be non-intrusive yet informative. It features:
- **Search Highlighting**: Highlights the characters you've typed in the chord list.
- **Icon Support**: Use Blender's built-in icons to quickly identify actions.
- **Custom Themes**: Change the overlay's colors, transparency, and position.

## Overlay Components

- **Header**: Shows current chord buffer
- **Body**: Lists available next tokens
- **Footer**: Shows additional information
