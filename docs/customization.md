# Customization

## Preferences

- **Config Path**: Path to the configuration file. See [Import & Export](configuration/import_export.md) for details about configuration file management.
- **Scripts Folder**: Default directory for Python scripts used in script mappings.

See [Overlay Configuration](customization/overlay.md) for overlay settings.

### Autosave

Changes are saved automatically after 5 seconds of inactivity. The timer resets on each change, so rapid edits only trigger one save. See [Autosave](configuration/autosave.md) for detailed information.
