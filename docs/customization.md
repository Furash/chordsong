# Customization

## Preferences

- **Config Path**: Path to the configuration file.
- **Scripts Folder**: Default directory for Python scripts used in script mappings.

See [Overlay Configuration](customization/overlay.md) for overlay settings.

## Configuration Management

Configuration is stored in a single JSON file: `.../scripts/presets/chordsong/chordsong.json`

You can set a custom **Config Path** in preferences. Use **Load** and **Save** to import/export configs or switch profiles.

### Autosave

Changes are saved automatically after 5 seconds of inactivity. The timer resets on each change, so rapid edits only trigger one save.
