# Customization

Make Chord Song truly yours by customizing its behavior and appearance.

## Preference Settings

### UI Settings
Modify the look and feel of the overlay:
- **Font Size**: Adjust the text size for better readability.
- **Background Color**: Change the overlay's background color and opacity.
- **Item Padding**: Control the spacing between items in the list.
- **Positioning**: Choose where the overlay appears on your screen.

### Behavior Settings
- **Timeout**: Set the duration the overlay remains visible after inactivity.
- **Auto-Execute**: Enable or disable automatic execution of chords when a unique match is found.
- **Recent Items**: Configure how many recent items are tracked and displayed.
